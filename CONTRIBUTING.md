# Contributing to AceBlocks

Thank you for your interest in contributing to AceBlocks!

AceBlocks is an educational platform designed to help people learn coding through block-based programming and interactive lessons.

## How to Contribute

There are numerous ways you can help improve AceBlocks:

### Report Bugs

If you find a bug, please open an issue and include:

- A clear description of the problem
- Steps to reproduce the issue
- Screenshots if possible

### Suggest Features

If you have ideas for new features or improvements:

- Open a feature request in the Issues section
- Clearly explain the idea and why it would help AceBlocks

### Submit Code

1. Fork the repository
2. Create a new branch
3. Setup tooling
   1. Install node, and npm https://nodejs.org/en/download
   2. Install tools using `npm ci`
4. Make your changes
5. Test your changes
   - Run `tsc --watch` to test TypeScript code while making changes
   - Run a web server of your choice to test everything working together
6. Submit a Pull Request

## Coding Guidelines

Please follow these guidelines when contributing code:

- Keep code clean and readable
- Comment important sections
- Test your changes before submitting
- Do not break existing features

## Code of Conduct

By contributing to this project, you agree to follow our Code of Conduct.

## Questions

If you have questions, open an issue or contact the project maintainers.

Thank you for helping improve AceBlocks!
