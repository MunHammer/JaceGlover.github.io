The Platform

What is AceBlocks?

AceBlocks is a visual coding platform designed to help students learn programming concepts through block-based coding, while also introducing real scripting — including Luau for Roblox development. It bridges the gap between beginner-friendly visuals and actual code, so students build real skills from day one.

The Reason

Why AceBlocks?

  • Beginner-friendly block coding system
  • Bridges into real-world coding (Luau)
  • Designed for schools and classrooms
  • Supports creative game development
